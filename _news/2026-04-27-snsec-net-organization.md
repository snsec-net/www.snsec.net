We have created the **[snsec-net](https://github.com/snsec-net)** GitHub organization to share artifacts and source code from our published studies. The first artifact is the source code of **DRIFT: Drift-Resilient Invariant-Feature Transformer for DGA Detection** (IEEE/IFIP DSN'26), now hosted at **[snsec-net/2026-DSN-DRIFT](https://github.com/snsec-net/2026-DSN-DRIFT)**. The source of this website also lives in the organization at **[snsec-net/www.snsec.net](https://github.com/snsec-net/www.snsec.net)**.
